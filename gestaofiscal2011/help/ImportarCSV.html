Importar CSV


A ferramenta Importar CSV, importa arquivos no formato ".csv" de base de dados do IPTU e base de dados do SIF para aplicação. O arquivo ".csv" de base de dados do SIF precisa conter a coluna Chave Geo para ser importado. As informações dos arquivos serão exibidas na Lista de Resultados.


I


Camada: exibe camadas a qual o arquivo ".csv" importado pertence;

Selecionar Arquivo: disponibiliza caixa de dialogo para seleção do arquivo ".csv";

Fechar: fecha janela Importar CSV;

Minimizar: minimiza janela Importar CSV.
